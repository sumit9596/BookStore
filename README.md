# BookStore 📚

A modern online bookstore application with a React frontend and Node.js backend.

## 🚀 Live Project
**[https://book-store-alpha-mocha.vercel.app/](https://book-store-alpha-mocha.vercel.app/)**

## 📋 Project Structure

```
BookStore/
├── Frontend/my-project/     - React + Vite frontend
├── Backend/                 - Node.js + Express backend
├── DEPLOYMENT.md            - Deployment guide
└── vercel.json              - Vercel configuration
```

## 🛠️ Tech Stack

### Frontend
- React 18
- Vite
- Tailwind CSS
- Axios (API calls)

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## 📦 Features

- Browse and search books
- User authentication (Sign Up / Login)
- Shopping cart functionality
- Responsive design
- Real-time data updates

## 🚀 Deployment

The project is deployed on **Vercel**:
- Frontend: Hosted on Vercel
- Backend: API endpoints via Vercel serverless functions

For detailed deployment instructions, see [DEPLOYMENT.md](DEPLOYMENT.md)
